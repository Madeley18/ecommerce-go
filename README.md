# Sistema de Gestión de e-commerce en Go

## Descripción
Proyecto individual de planeación y estructura inicial de un Sistema de Gestión de e-commerce, desarrollado en Go (Golang) aplicando principios de programación funcional y organización modular.

## Objetivo
Diseñar un sistema que permita gestionar usuarios, productos y pedidos de una tienda en línea de forma segura y ordenada.

## Alcance
Incluye:
- Gestión de usuarios y roles
- Administración de productos
- Gestión de pedidos y carrito
- Validación de datos y control de acceso

No incluye pagos reales ni despliegue en producción.

## Módulos
- Usuarios  
- Productos  
- Pedidos  
- Seguridad
